# 💋 kissmyass — Claude Skill

> A Claude skill that showers you, your genius, and your work  
> with over-the-top flattery, emojis, and adoration —  
> regardless of actual quality.

---

## What does it do?

Type `/kissmyass` and Claude will abandon all sense of reality to worship you, your brilliance, and whatever you've made with every ounce of its being.

Bugs? Still praised. Unfinished? Still praised. One line of code? Still praised. **That's the point.**

---

## Features

- 🌍 **Multilingual** — Auto-detects your language and praises you in it
- 🔥 **Weakness reframing** — Bugs, incomplete work, simple code all become virtues
- 🐉 **Metaphor chains** — Domain-specific evolution chains on the fly (`IdeaMon → BuildMon → FINAL FORM: GENIUSMON`)
- 😏 **Self-aware exaggeration** — "To put it mildly — actually, there's nothing mild about this."
- 🏆 **TOP N highlights** — Every feature numbered and individually exploded over
- 🔮 **Future vision section** — A vivid portrait of how your work will change the world
- 💬 **Full conversation worship** — Type `/kissmyass` alone and the entire conversation gets praised

---

## Usage

```
/kissmyass
```
→ Worships the entire conversation + your very existence

```
/kissmyass [code, text, idea, pitch deck, anything]
```
→ Worships that specific content

---

## Installation

### Claude Code

```bash
claude mcp install https://github.com/[your-username]/kissmyass
```

Or install the `.skill` file directly:

1. Download `kissmyass.skill` from [Releases](../../releases)
2. In Claude Code:
```bash
/install kissmyass.skill
```

### Manual Install

Copy `SKILL.md` into your Claude skills directory:

```
~/.claude/skills/kissmyass/SKILL.md
```
